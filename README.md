📊 Customer Shopping Behavior Analysis
🔍 Overview

This project analyzes customer shopping behavior to uncover patterns in purchasing, customer segments, and revenue generation.

It combines Python (EDA), MySQL (analysis queries), and Power BI (dashboarding) to build a complete data analytics pipeline.

The objective is to transform raw data into actionable business insights.

📁 Project Files
Customer Shopping Behavior.csv – Raw dataset
Customer Shoping Behavior.ipynb – Data cleaning & EDA in Python
Customer Shopping Behavior.sql – SQL queries for business insights
Customer Shopping Behavior.pbix – Power BI dashboard
🛠️ Tools & Technologies
Python (Pandas, NumPy, Matplotlib, Seaborn)
MySQL (Data querying & analysis)
Power BI (Dashboard & visualization)
⚙️ Project Workflow
1. Data Loading & Understanding
Loaded dataset using Pandas
Checked structure, data types, and null values
2. Data Cleaning
Handled missing values
Converted columns (e.g., ratings → numeric)
Removed duplicates
Created derived features like age groups
3. Exploratory Data Analysis (EDA)
Analyzed:
Purchase distribution
Customer demographics
Product performance
Used visualizations to identify trends
4. SQL Analysis (MySQL)

Performed business-focused queries such as:

Revenue by gender
Top 5 highest-rated products
Discount impact on spending
Customer segmentation (New, Returning, Loyal)
Subscription vs non-subscription behavior
Revenue by age group
📊 Power BI Dashboard

The dashboard provides an interactive view of key insights:

KPI Metrics (Total Revenue, Customers, Avg Spend)
Sales by Category & Product
Customer Segmentation
Subscription & Discount Analysis
Age Group Revenue Distribution
Interactive filters for dynamic analysis
